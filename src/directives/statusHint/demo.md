# 状态样式指令

## 何时使用
展示订单状态

## 代码演示

```js
/*angular*/
/*desc 状态样式 desc*/

<span status-hint type="c-green" >完成</span>
<span status-hint type="c-red" >失败</span>
<span status-hint type="c-blue" >待完成</span>
<span status-hint type="c-grey" >已关闭</span>
```

## API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| type | 样式 | string |  |
