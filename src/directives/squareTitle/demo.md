# 方块标题指令

## 何时使用
展示标题

## 代码演示

```js
/*angular*/
/*desc 方块标题 desc*/

<span square-title type="large" cor="#7b7de5">巨划算</span>
<span square-title type="middle" cor="#7b7de5">巨划算</span>
<span square-title type="min" cor="#7b7de5">巨划算</span>
```

## API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| type | 样式大小类型：large, middle, min | string |  |
| cor | 背景色：自定义 | string |  |
